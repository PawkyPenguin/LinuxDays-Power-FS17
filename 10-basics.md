### Automatic updating and shutdown

```
$ sudo su
root's password:
$ zypper ref && zypper -n up && poweroff
```

* `sudo su`: Become root
* `zypper ref`: Update repository
* `zypper -n up`: Install upgrades
* `-n` stands for `no confirmation`
* `A && B`: Execute B if A completed successfully
